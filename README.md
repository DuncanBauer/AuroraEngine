### Naming Conventions
```
Classes:
- Capitalized
- CamelCase

Functions:
- Capitalized
- CamelCase

Variables:
- Uncaptialized
- CamelCase
    Static:
        - Begins with s_
    Pointer:
        - Begins with p_
    Reference:
        - Begins with r_

    Class member:
        - Beings with m_
        Static:
            - Begins with sm_
        Pointer:
            - Begins with pm_
        Reference:
            - Begins with rm_
```
